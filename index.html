🎲 Amostragem Casual Simples (ACS)

📌 Definição:

Nesse tipo de amostragem, cada elemento da amostra é escolhido por sorteio individual a partir da população. Normalmente utilizamos uma porcentagem da população total.

🔢 Amostragem Sistemática

📌 Definição:

Método que utiliza uma "semente" sorteada e um intervalo fixo (salto) para selecionar elementos de forma sistemática. Reduz a quantidade de sorteios necessários.

📊 Amostragem Proporcional Estratificada

📌 Definição:

Técnica para populações divididas em estratos (subgrupos), garantindo representatividade proporcional de cada grupo na amostra final.
